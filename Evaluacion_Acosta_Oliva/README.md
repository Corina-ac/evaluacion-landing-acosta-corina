**Estudiante:** Corina Alejandra Acosta Oliva

**Nombre de la landing page:** NexoPlay  
**Paleta de colores:** Morado (#6A0DAD), Azul eléctrico (#0055FF), Negro (#000000)

## Descripción

NexoPlay es una landing page diseñada para una tienda digital de videojuegos. El sitio presenta un diseño moderno y responsivo que permite a los usuarios explorar productos, conocer servicios, ver una galería multimedia y contactar con la tienda.

## Instalación y ejecución

1. Clonar el repositorio o descargar el archivo ZIP.
2. Abrir el archivo `index.html` en cualquier navegador web moderno.
3. Navegar por las diferentes secciones mediante el menú superior.

## Estructura del proyecto

```
Evaluacion_Acosta_Oliva/
├── index.html                    # Página principal
├── README.md                     # Documentación del proyecto
├── css/
│   ├── general.css               # Estilos generales y variables
│   └── index.css                 # Estilos específicos y media queries
├── img/                          # Recursos gráficos
├── multimedia/                   # Recursos audiovisuales
├── pages/
│   ├── contacto/contacto.html    # Página de contacto
│   ├── galeria/galeria.html      # Página de galería
│   ├── servicios/servicios.html  # Página de servicios
│   └── nosotros/nosotros.html    # Página sobre nosotros
```

## Tecnologías utilizadas

- HTML5 (etiquetas semánticas)
- CSS3 (Flexbox, Grid, variables, transiciones)
- Bootstrap 5.3 (CDN)
- Font Awesome 6.5.0 (CDN)
- Diseño mobile-first con 3 media queries
